Простота использования

Для работы с сервисом вам понадобится ОС Linux и веб-браузер с установленным Java плагином. Интуитивно понятный веб-интерфейс с которым может справится любой системный администратор с любой квалификацией.

Безопасность

Сервис WebVirtMgr использует для передачи данных шифрованное соединение. Сервис получает доступ только к гипервиозору KVM через библиотеку libvirt, а не ко всей ОС что повышает безопасность сервера в целом.

Возможности

Полный контроль над виртуализацией. Создание, выключение, установка, снапшоты - все это и не только вы можете делать с виртуальными машинами. Все ваши действия будут фиксироваться в журнале событий.
