Web Virtual Manager


WebVirtMgr сервис для управления виртуальными серверами с помощью браузера. В качестве основных технологий используються: гипервизор KVM и библиотека управления виртуализацией libvirt. Созадние новых виртуальных машин, конфигурирование и управление их ресурсами. Графический доступ к вашей виртуальной машние предоставляется с помощью технологии VNC через SSH туннель.


Зарегистрироваться

С чего начать?


1. Зарегистрироваться и подтвердить регистрацию.

2. Установить и настроить libvirt и KVM на вашем сервере.

3. Войти и добавить ваш сервер для его управления.
